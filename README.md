# 💰 FinanceApp — Controle de Gastos Pessoais

App completo para controle de finanças pessoais, desenvolvido com Node.js, React e TypeScript.

## 🚀 Stack

- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript
- **Banco de dados:** MySQL (Docker)
- **ORM:** Prisma

## 📦 Módulos

- 🔐 Autenticação (Login/Cadastro)
- 🏠 Dashboard com resumo mensal
- 💳 Gerenciamento de Contas
- 📥 Receitas e 📤 Despesas
- 🏷️ Categorias personalizadas
- 📊 Relatórios com gráficos

## ⚙️ Como rodar localmente

### Pré-requisitos
- Node.js v22+
- Docker e Docker Compose

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
